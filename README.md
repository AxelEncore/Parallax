Просто порт Axiom на NeoForge без надобности установки костылей в виде Connector

На данный момент поддерживается 1.21.1 версия
Есть полностью исходный код, для того, чтобы вы могли создавать что-то свое, обновлять мод и не думать о декомпиляции
Мод полностью на 100% переведен на русский язык без странной мешанины английско/русского
Исправлены небольшие баги при использовании Axiom на сервере, когда отображались "фантомные блоки" только у клиента, что мешало уверенно строить
Теперь обучение и подсказки, которые не дают пользоваться инструментами без прохождения всех этапов можно пропустить сразу при запуске Axiom - вас будут спрашивать пройти обучение да/нет
Исправлена повторная загрузка ресурсом minecraft при запуске игры с Axiom
Решение полностью переписать код на NeoForge вызвано тем, что разработчик оттаргает Forge, не предоставляет исходников, для написания порта, а так же абсолютно игнорирует ошибки, которые всплыли при использовании не Fabric, а NeoForge + Connector
